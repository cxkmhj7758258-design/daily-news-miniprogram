# 📰 每日新闻早报 — 微信小程序

一个每天早上定时推送昨日精选新闻的微信小程序，用户可自由选择关注的领域。

## 功能

- 📰 **每日早报** — 每天推送你关注的领域新闻精选
- ⚙️ **自定义关注** — 从时政、财经、科技等9个领域中自由选择
- 🔔 **定时推送** — 每天早上8点微信服务通知推送
- 📱 **轻量体验** — 无需下载App，微信内直接使用

## 技术栈

- 微信小程序原生（WXML + WXSS + JS）
- 微信云开发（云数据库 + 云函数 + 定时触发器）
- 微信订阅消息（每日推送）

## 项目结构

```
├── app.json                    # 全局配置
├── app.js                      # 全局逻辑
├── app.wxss                    # 全局样式
├── project.config.json         # 项目配置
├── images/tabbar/              # 导航图标
├── pages/
│   ├── index/                  # 首页 - 每日早报
│   ├── settings/               # 关注设置
│   ├── article/                # 新闻详情
│   └── history/                # 我的
├── components/
│   └── news-card/              # 新闻卡片组件
└── cloudfunctions/
    ├── aggregateNews/          # 新闻聚合云函数
    ├── dailyPush/              # 每日定时推送云函数
    └── userManager/            # 用户管理云函数
```

## 快速开始

1. 在微信开发者工具中导入项目
2. 填写 `project.config.json` 中的 `appid`
3. 开通云开发环境，创建以下集合：
   - `users` — 用户偏好
   - `news` — 新闻数据
   - `pushLogs` — 推送日志
   - `readLogs` — 阅读记录
4. 上传并部署所有云函数
5. 在微信小程序后台申请订阅消息模板
6. 修改 `dailyPush/index.js` 中的 `templateId`
